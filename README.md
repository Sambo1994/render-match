# render-match
Ai render plugin for sketchup - WORKING

HOW TO INSTALL.
1.Download - RenderMatch.rbz
2.in sketchup - extension manager - install exetension - add RenderMatch.rbz
3.Add API Key 
You need a Stability AI API key first. Get one at https://platform.stability.ai → sign in → account/API keys section → generate a key. Paste that into the prompt box when it appears, and it'll be saved locally so you won't be asked again (until you click "Reset API Key").
4.Done
